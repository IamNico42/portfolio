# 1. Was bedeutet Nebenläufigkeit?

**Nebenläufigkeit** bedeutet:

> Mehrere Abläufe passieren gleichzeitig oder überlappend.

Beispiel:
```
Client A sendet AnfrageClient B sendet AnfrageClient C sendet AnfrageServer bearbeitet mehrere Dinge gleichzeitig oder zeitlich überlappend
```

In verteilten Systemen ist das normal, weil viele Clients, Dienste und Komponenten gleichzeitig aktiv sein können.

Einfaches Beispiel
```
100 Nutzer öffnen gleichzeitig deine Website.
```

Dann will man nicht
```
Nutzer 1 wird komplett fertig bearbeitetdanach Nutzer 2danach Nutzer 3...
```

Das wäre extrem langsam.
Man möchte:
```
mehrere Anfragen gleichzeitig / überlappend bearbeiten
```
